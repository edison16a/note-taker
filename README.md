# Note Taker (Real-Time Voice Transcription)

https://edison-note-taker.vercel.app/

A simple, local-first note taker that uses **Chrome’s built-in speech recognition** to create **real-time transcriptions** while you’re in class.

You can:
- Create **class tabs** (ex: Gov, Stats, Lit)
- Automatically get a **daily session tab** for today (rename it if you want)
- Hit **Listen** and it transcribes everything it hears
- Everything saves **locally in your browser** (`localStorage`) — no account, no backend
- Download any session as a **.txt** file
